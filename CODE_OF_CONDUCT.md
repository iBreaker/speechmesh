# Code of Conduct

SpeechMesh aims to be a professional, respectful, and technically rigorous project.

## Expected Behavior

- be respectful and direct
- assume good intent while reviewing critically
- focus criticism on ideas, code, and architecture rather than people
- communicate clearly about risks, tradeoffs, and unknowns
- help keep the project usable for operators, SDK consumers, and contributors

## Unacceptable Behavior

- harassment or personal attacks
- abusive, discriminatory, or intentionally hostile language
- doxxing or sharing private information without permission
- disruptive behavior that blocks healthy technical collaboration

## Enforcement

Project maintainers may remove, edit, or reject contributions and interactions that violate these standards.
