# Documentation

This directory contains the canonical project documentation for SpeechMesh.

## Core Guides

- `architecture.md` - system model, runtime shapes, extension points
- `websocket-protocol.md` - client/gateway wire contract for WebSocket v1
- `deployment.md` - local, split-runtime, and Kubernetes deployment guidance
- `sdk-guide.md` - how to consume the Rust and Go client SDKs
- `testing.md` - validation matrix, local checks, and live E2E commands
- `compatibility.md` - protocol and SDK compatibility expectations
- `roadmap.md` - project status and next delivery milestones
